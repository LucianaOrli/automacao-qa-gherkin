🚀 O Projeto Luxbyor
Este repositório apresenta uma solução robusta de automação de testes desenvolvida sob a metodologia Luxbyor, utilizando Python, Playwright e Pytest-BDD. O foco é demonstrar a aplicação de boas práticas de engenharia de software (Page Object Model) e a resiliência de scripts em ambientes de CI/CD.

 🛠️ Tecnologias Utilizadas
Linguagem: Python 3.11+
Framework de Teste: Pytest-BDD (Behavior Driven Development)
*Engine de Automação: Playwright (Chromium)
CI/CD: GitHub Actions
Relatórios: Pytest-HTML

🏗️ Arquitetura do Projeto (POM)
O projeto segue o padrão **Page Object Model**, separando as responsabilidades para facilitar a manutenção:
 `features/`: Especificações dos cenários em Gherkin.
`testes/agiblog_page.py`: Mapeamento de elementos e ações da página (Locators).
 `testes/test_agiblog.py`: Runners e definições de passos (Steps).
